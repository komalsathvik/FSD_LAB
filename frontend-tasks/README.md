# Frontend Tasks

This folder contains four Bootstrap 5 frontend assignments implemented at a basic, accurate level according to `Tasks.pdf`.

## Projects

1. `student-management/` - Responsive Student Management Portal
2. `course-registration/` - Online Course Registration Portal
3. `college-event-management/` - College Event Management System
4. `portfolio/` - Personal Portfolio Website

Open each project's `index.html` in a browser to start.
