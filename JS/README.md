# JavaScript Basic Practice

This folder has small beginner-level examples for common JavaScript topics.

## Files

- `callbacks.js` - passing a function into another function
- `promises.js` - basic Promise with `.then()`, `.catch()`, and `.finally()`
- `async-await.js` - using `async` and `await` with a Promise
- `timers.js` - examples of `setTimeout` and `setInterval`
- `modules/app.js` - imports functions from another file
- `modules/mathUtils.js` - exports simple functions
- `index.html` - browser page for trying the module example

## How to Run

For normal files, use Node.js:

```bash
node callbacks.js
node promises.js
node async-await.js
node timers.js
```

For modules, open `index.html` in the browser and check the browser console.
