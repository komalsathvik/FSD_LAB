# Node Built-in Module Practice

Run any example with:

```bash
node filename.js
```

Examples:

```bash
node events.js
node fs.js
node readstream.js
node os.js
node http.js
```

For the responsive form example:

```bash
node form-server.js
```

Then open:

```text
http://localhost:3000
```

When the form is submitted, the data is saved in `form-data.txt`.
